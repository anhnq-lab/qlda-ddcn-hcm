# Cấu hình dự án qlda-ddcn-hcm

## ⚠️ RULES (BẮT BUỘC)
- **LUÔN chạy local ở http://localhost:5000** — KHÔNG được đổi port sang giá trị khác
- Khi mở browser để test, luôn dùng URL: `http://localhost:5000`
- Khi khởi động dev server, dùng: `npm run dev` (port 5000 đã cấu hình sẵn trong `vite.config.ts`)

## Dev Server
- **Port**: 5000 (cố định, không đổi)
- **URL**: http://localhost:5000
- **Host**: 0.0.0.0 (cho phép truy cập từ mạng LAN)
- **Lệnh chạy**: `npm run dev`
