Checklist chuẩn bị trước buổi demo

🔧 Kỹ thuật

  • Internet ổn định tại địa điểm demo
  • App đã mở, đã đăng nhập sẵn
  • Upload sẵn file IFC cho BIM 3D
  • Test tất cả tabs (không lỗi 400/500)
  • Test AI chatbot, dark mode, Ctrl+K
  • Data: 3 dự án, 5 HĐ, nhân sự có đủ

💻 Thiết bị

  • Laptop sạc đầy + adapter
  • HDMI/USB-C cho máy chiếu
  • Chuột rời (BIM cần zoom/rotate)
  • Resolution ≥ 1920×1080

📄 Tài liệu

  • In tài liệu giới thiệu DOCX
  • Name card / thông tin liên hệ
  • File IFC mẫu dự phòng
